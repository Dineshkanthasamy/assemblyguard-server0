🚀 Complete Deployment Guide
STEP 1 — Deploy Cloud Server (Render.com — Free)
1. Create account at render.com
2. New → Web Service → Connect GitHub repo
3. Upload the server/ folder
4. Build: npm install
5. Start: node server.js
6. Copy your URL → e.g. https://assemblyguard-abc.onrender.com
STEP 2 — Configure Jetson AI Script
Open jetson_ai.py
Set: CLOUD_URL = "https://YOUR-APP.onrender.com"
Set: CLASS_NAMES to match your YOLOv8 data.yaml
Set: REQUIRED_PARTS for your assembly rule
STEP 3 — Run on Jetson Nano
sudo nvpmodel -m 0
sudo jetson_clocks
python3 jetson_ai.py
STEP 4 — Open Dashboard Anywhere
Open https://YOUR-APP.onrender.com
on any phone, tablet or laptop
Login with your credentials
✓ See live station status in real time
